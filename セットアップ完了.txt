✅ セットアップ内容
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Python仮想環境 (venv) の構築
   - 場所: .\venv\
   - Python バージョン: 3.14

2. インストール済みライブラリ
   ┌─────────────────────────────────────────────┐
   │ streamlit      1.50.0  (ダッシュボード)     │
   │ pandas         2.3.3   (データ分析)         │
   │ numpy          2.3.4   (数値計算)           │
   │ plotly         6.4.0   (グラフ作成)         │
   │ matplotlib     3.10.7  (グラフ作成)         │
   │ seaborn        0.13.2  (統計可視化)         │
   │ openpyxl       3.1.5   (Excel操作)          │
   │ pillow         11.3.0  (画像処理)           │
   └─────────────────────────────────────────────┘

3. 作成されたファイル
   ┌─────────────────────────────────────────────┐
   │ app.py              - メインアプリケーション │
   │ requirements.txt    - ライブラリ一覧         │
   │ README.md           - プロジェクト説明       │
   │ run_dashboard.bat   - 起動用バッチファイル   │
   │ .gitignore          - Git除外設定            │
   └─────────────────────────────────────────────┘

  🚀 ダッシュボードの起動方法
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【最も簡単な方法】
  run_dashboard.bat をダブルクリック

【コマンドラインから】
  venv\Scripts\python.exe -m streamlit run app.py

起動後、ブラウザで以下のURLにアクセスできます：
  → http://localhost:8501
